# Games

ブラウザで遊べるシンプルなHTMLゲーム集です。

## 起動方法

`index.html` をブラウザで開くと、ゲーム一覧から各ゲームを起動できます。

## ゲーム一覧

- `game1.html`: スイカゲーム風ゲーム
- `game2.html`: テトリス風ゲーム
- `game3.html`: ボンバーマン風ゲーム
- `game4.html`: 30マスすごろく
- `game5.html`: パックマン風ゲーム
- `game6.html`: 対戦型 戦艦ゲーム
- `game7.html`: じゃんけん ﾌｧｲﾀｰ
- `game8.html`: 危機一髪風ゲーム
