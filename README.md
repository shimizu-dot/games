# Games

ブラウザで遊べるシンプルなHTMLゲーム集です。

## 起動方法

`index.html` をブラウザで開くと、ゲーム一覧から各ゲームを起動できます。

## ゲーム一覧

- `game1.html`: スイカゲーム
- `game2.html`: テトリス
- `game3.html`: ボンバーマン風ゲーム
- `game4.html`: 30マスすごろく

## ファイル構成

- `index.html`: ゲーム一覧ページ
- `game1.html`: スイカゲーム本体
- `game2.html`: テトリス本体
- `game3.html`: ボンバーマン風ゲーム本体
- `game4.html`: 30マスすごろく本体
- `README.md`: この説明ファイル
